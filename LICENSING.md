## Audio samples

The runtime sample set is distributed as a GitHub Release asset
(`dataSonification-runtime-samples.zip`) on the
[dataSonification-java](https://github.com/maestrovt/dataSonification-java/releases)
repository rather than tracked in git. Its contents and per-set acknowledgments
are documented in the `READHERE.txt` inside that archive. In particular, the
**ACB (Acoustic Branding)** sample set is redistributed with grateful
acknowledgment to its creators.

## General MIDI soundbank

`soundbank-deluxe.gm` is **not** distributed. It is only used by the
MIDI-instrument playback path (not the sample-based instruments), and its
provenance is uncertain. Supply your own General MIDI soundbank if you need it.

## The Java engine

The companion audio engine lives in a separate repository
([dataSonification-java](https://github.com/maestrovt/dataSonification-java))
and has its own `LICENSING.md` covering the commercial libraries (JMSL, JSyn)
it requires.
